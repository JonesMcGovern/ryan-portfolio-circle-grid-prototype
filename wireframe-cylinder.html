Playground 01

Interactive Soda Can

This proof of concept explores how AI-generated code can quickly create interactive 3D experiences. Using a simple soda can with subtly beveled ends, I focused on demonstrating the workflow rather than building a production-ready asset. A toggle switches between a wireframe view and the finished labeled can, highlighting how platforms like Codex can generate interactive 3D objects from straightforward prompts.
